# PP2

This repository contains practical assignments and study materials for the PP2 course.

The project includes structured practice tasks covering core and advanced Python concepts such as functions, lambda expressions, object-oriented programming, and inheritance.

All examples are organized into separate practice folders to demonstrate progressive learning and understanding of the subject.

Each folder contains Python examples and exercises related to specific topics studied during the course.

## Purpose

The purpose of this repository is to demonstrate practical skills in Python programming and the ability to organize and structure code effectively.
