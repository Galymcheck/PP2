Python Pygame Projects Collection

This repository contains three game projects developed using Python and Pygame as part of a learning practice in game development and event-driven programming.

Snake Game A classic Snake game where the player controls a snake, eats food, grows in size, and tries to avoid collisions.

Racing Game A simple car racing game where the player avoids enemy cars coming from the top of the screen.

Paint Application A mini paint program that allows drawing shapes and freehand sketches using different tools.
